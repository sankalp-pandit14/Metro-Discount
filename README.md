# Metro-Discount
A simple web-based application built using HTML, CSS, and JavaScript that generates and applies metro discount codes for users.

📌 Features

🎟️ Generate random discount codes

💸 Apply discount on metro fare

⚡ Instant calculation of discounted price

🧑‍💻 Simple and user-friendly interface

📱 Responsive design (works on mobile & desktop)

🛠️ Tech Stack

HTML – Structure of the webpage

CSS – Styling and layout

JavaScript – Logic for discount generation and calculation
